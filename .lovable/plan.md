

# টপবার রিডিজাইন — রেফারেন্স ইমেজ অনুযায়ী

## রেফারেন্স বিশ্লেষণ
ছবিতে একটি **ডার্ক নেভি/প্রাইমারি ব্যাকগ্রাউন্ড** টপবার দেখা যাচ্ছে যেখানে:
- **বামে**: কোম্পানি নাম (Softifybd Ltd) + সার্চ আইকন + "Search Customer" ড্রপডাউন
- **ডানে**: "Open Support Ticket" বাটন + "Online Client Monitoring" বাটন + Notification bell (badge সহ) + Info icon (badge সহ) + User avatar

## প্রস্তাবিত ডিজাইন (আরও সুন্দর ও লজিক্যাল)

### লেআউট:
```text
┌──────────────────────────────────────────────────────────────────┐
│ 🏢 Company Name  🔍 Search Customer ▼  │  📧 Support  🖥 Online Monitor  🔔⁶ ℹ️⁶ 👤 │
└──────────────────────────────────────────────────────────────────┘
```

### স্টাইল:
- **ব্যাকগ্রাউন্ড**: `bg-primary` (ডার্ক নেভি/থিম কালার) — সাদা ব্যাকগ্রাউন্ডের বদলে
- **টেক্সট/আইকন**: সাদা (`text-primary-foreground`)
- কম্প্যাক্ট হাইট: `h-12` (বর্তমান `h-16` থেকে ছোট)

### বামের আইটেম:
1. **কোম্পানি নাম** — TenantSwitcher-এর বদলে শুধু টেন্যান্ট নাম দেখাবে (non-superadmin দের জন্য)
2. **Search Customer** — সিলেক্ট/কম্বোবক্স স্টাইলে (ছবির মতো ড্রপডাউন)

### ডানের আইটেম:
1. **Open Support Ticket** — আইকন + টেক্সট বাটন
2. **Online Client Monitoring** — আইকন + টেক্সট বাটন
3. **Notification Bell** — ব্যাজ সহ
4. **Info/Alert** — ব্যাজ সহ
5. **User Avatar** — প্রোফাইল ড্রপডাউন
